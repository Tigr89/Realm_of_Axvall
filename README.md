# Realm of Axvall
Ett skolprojekt där vi tillsammans försöker bygga ett spel som efterliknar Realm of the Mad God.

## Viktigt att tänka på!
* Se till att skapa en egen branch och att ni alltid arbetar i den.
* Ni mergar till main enbart när ni testat färdigt den mekanik/uppgift ni jobbat på.
